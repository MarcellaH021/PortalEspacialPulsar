- Descrição

O Portal Espacial Pulsar é uma aplicação do tipo Single Page Application (SPA) desenvolvida em React, que consome dados da API pública da NASA para exibir conteúdos astronômicos de forma interativa e educativa.

O sistema vai simula um portal institucional de divulgação científica, permitindo aos usuários explorar imagens espaciais, visualizar detalhes e salvar conteúdos favoritos, promovendo aprendizado e engajamento com temas de ciência e tecnologia.
